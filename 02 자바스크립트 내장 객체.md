### 1. Object 객체

- Object 객체는 모든 자바스크립트 객체의 루트 객체입니다.

```javascript
// 빈 객체 생성
let person = new Object();

// 멤버 설정
person.firstName = "채은";
person.lastName = "임";
person.age = 27;
person.getFullName = function () {
  return this.lastName + this.firstName;
};

console.log(person.getFullName()); // 임채은
```

### 2. String 객체

- String 객체는 자바스크립트에서 문자열을 다루기 위한 객체로 문자열을 다룰 때 유용한 프로퍼티와 함수를 제공합니다.

  | 함수                                      | 정의                                                   |
  | :---------------------------------------- | :----------------------------------------------------- |
  | length                                    | 문자열의 길이 반환                                     |
  | indexOf(특정 문자열)                      | 제일 처음에 발견된 특정 문자열이 시작되는 index 반환   |
  |                                           | 특정 문자열 없으면 -1 반환                             |
  | lastIndexOf(특정 문자열)                  | 제일 마지막에 발견된 특정 문자열의 index 반환          |
  |                                           | 특정 문자열 없으면 -1 반환                             |
  | slice(시작 인덱스, 종료 인덱스)           | 시작 인덱스에서 종료 인덱스 전까지 잘라내서 반환       |
  |                                           | 인덱스 음수 가능                                       |
  | substring(시작 인덱스, 종료 인덱스)       | slice()와 동일                                         |
  |                                           | 인덱스 음수 불가능                                     |
  | substr(시작 인덱스, 잘라낼 문자열의 길이) | 시작 인덱스에서 잘라낼 문자열의 길이만큼 잘라내서 반환 |
  |                                           | 인덱스 음수 가능                                       |
  | replace(특정 문자열, 지정한 문자열)       | 특정 문자열을 지정한 문자열로 변환                     |
  |                                           | 처음 발견된 특정 문자열만 바꿈                         |
  |                                           | 특정 문자열이 `/Seoul/i` 일 경우, 대소문자 구분 X      |
  |                                           | 특정 문자열이 `/Seoul/g` 일 경우, 모든 문자열          |
  | toUpperCase()                             | 모두 대문자 변경                                       |
  | toLowerCase()                             | 모두 소문자 변경                                       |
  | concat(문자열1, 문자열2, ...)             | 2개 이상의 문자열을 하나의 문자열로 합침               |
  | trim()                                    | 문자열의 앞,뒤 공백을 모두 제거                        |
  | padStart(지정된 길이, 추가할 문자)        | 문자열 앞에 추가할 문자를 지정된 길이만큼 추가         |
  | padEnd(지정된 길이, 추가할 문자)          | 문자열 뒤에 추가할 문자를 지정된 길이만큼 추가         |
  | charAt(특정 인덱스)                       | 특정 인덱스에 해당하는 문자 하나 반환                  |
  | charCodeAt(특정 인덱스)                   | 특정 인덱스에 해당하는 문자 하나의 유니코드 값 반환    |
  | split(특정 구분자)                        | 특정 구분자를 기분으로 문자열을 분리해서 배열로 반환   |
  | startsWith(특정 문자열)                   | 문자열의 시작이 특정 문자열로 시작되는지 확인          |
  | endsWith(특정 문자열)                     | 문자열의 끝이 특정 문자열로 종료되는지 확인            |

### 3. Number 객체

- Number 객체는 숫자를 다룰 때 유용한 프로퍼티와 함수를 제공합니다.

| 함수                          | 정의                                          |
| :---------------------------- | :-------------------------------------------- |
| toString()                    | 숫자형 데이터를 문자형 데이터로 반환          |
| toFixed(지정된 소수점 자릿수) | 지정된 소수점 자릿수까지 보여줌(반올림)       |
| toPrecision(길이)             | 정수와 소수를 포함한 길이 만큼 보여줌(반올림) |
| parseInt(문자열)              | 문자열의 시작이 숫자형이면 숫자형 데이터 반환 |
| parseFloat(문자열)            | 부동소수점 반환                               |

| 프로퍼티                 | 정의                                       |
| :----------------------- | :----------------------------------------- |
| Number.MAX_VALUE         | 자바스크립트에서 다룰 수 있는 최대 수      |
| Number.MIN_VALUE         | 자바스크립트에서 다룰 수 있는 가장 작은 수 |
| Number.POSITIVE_INFINITY | Infinity                                   |
| Number.NEGATIVE_INFINITY | -Infinity                                  |
| Number.NaN               | Not a Number                               |

### 4. Array 객체

| 함수                                                                     | 정의                                                                                           |
| :----------------------------------------------------------------------- | :--------------------------------------------------------------------------------------------- |
| toString()                                                               | 배열 안의 모든 문자를 쉼표를 이용해 모두 결합해서 하나의 문자열로 반환                         |
| join(지정한 문자)                                                        | 배열 안의 모든 문자를 지정한 문자를 이용해서 모두 결합해서 하나의 문자열로 반환                |
| pop()                                                                    | 배열에서 마지막 데이터를 제거하고, 마지막 데이터 반환                                          |
| push(새로운 요소)                                                        | 배열에서 새로운 요소를 추가                                                                    |
| shift()                                                                  | 배열에서 첫 번째 요소를 제거하고, 첫 번째 요소를 반환                                          |
| unshift(새로운 요소)                                                     | 배열의 맨 앞에 새로운 요소를 추가하고, 배열의 길이 반환                                        |
| splice(인덱스 번호, 삭제할 요소수, 새로운 요소1, 새로운 요소2, ...)      | 특정 인덱스 번호에서 삭제할 요소수 만큼 요소 삭제 후, 새로운 요소 추가                         |
| concat(배열1, 배열2, ...)                                                | 2개 이상의 배열을 하나의 배열로 결합                                                           |
| slice(시작 인덱스, 종료 인덱스)                                          | 시작 인덱스에서 종료 인덱스 전까지 잘라내서 배열 형태로 반환                                   |
| sort()                                                                   | 배열에 문자형 데이터가 있는 경우 오름차순 정렬                                                 |
| filter(callback(처리할 현재 요소, 처리할 현재 요소의 인덱스, 배열 전체)) | 배열에서 특정 조건을 만족하는 배열의 요소만을 찾아서 새로운 배열로 반환                        |
| map(callback())                                                          | 배열에 담긴 객체를 새로운 형태의 객체로 변환해서 배열로 반환                                   |
| reduce(callback(누적값, 현재 배열의 요소, 인덱스 번호))                  | 배열에 담긴 데이터를 하나씩 순회하며 callback 함수의 실행 값을 누적하여 결과값을 반환하는 함수 |

### 5. Date 객체

- Date 객체는 자바스크립트에서 날짜와 시간을 다루는 객체입니다.

```javascript
let now = new Date();
console.log(now); // Fri Mar 04 2022 23:34:25 GMT+0900 (한국 표준시)

let year = now.getFullYear(); // 4자리 년도
let month = now.getMonth(); // 0 ~ 11 사이의 월 정보
let day = now.getDate(); // 1 ~ 31 사이의 일 정보
let dayDay = now.getDay(); // 0 ~ 6 사이의 요일 정보
let hour = now.getHours(); // 0 ~ 23 사이의 시간 정보
let min = now.getMinutes(); // 0 ~ 59 사이의 분 정보
let sec = now.getSeconds(); // 0 ~ 59 사이의 초 정보

// 2022년 3월 4일 23시 34분 25초
console.log(`${year}년 ${month + 1}월 ${day}일 ${hour}시 ${min}분 ${sec}초`);
```

```javascript
let date = new Date(2021, 6, 24, 10, 33, 30, 0);
console.log(date); // Sat Jul 24 2021 10:33:30 GMT+0900 (한국 표준시)

date.setFullYear(2022); // 4자리 년도
date.setMonth(7); // 0 ~ 11 사이의 월 정보
date.setDate(25); // 1 ~ 31 사이의 일 정보
date.setHours(11); // 0 ~ 23 사이의 시간 정보
date.setMinutes(34); // 0 ~ 59 사이의 분 정보
date.setSeconds(31); // 0 ~ 59 사이의 초 정보
console.log(date); // Thu Aug 25 2022 11:34:31 GMT+0900 (한국 표준시)
```

### 6. Set 객체

- Set 객체는 유일한 값을 보장하는 값들의 집합입니다.

```javascript
let mySet = new Set();
```

| 함수                      | 정의                               |
| :------------------------ | :--------------------------------- |
| size                      | 데이터의 수                        |
| add(새로운 데이터)        | 새로운 데이터 추가                 |
| has(특정 데이터)          | 특정 데이터가 저장되어 있는지 확인 |
| delete(특정 데이터)       | 특정 데이터 삭제                   |
| clear()                   | 모든 데이터 삭제                   |
| forEach(callback(데이터)) | 모든 데이터 읽기                   |

### 7. Map 객체

- Map 객체는 키와 값을 맵핑시켜 값을 저장하며 저장된 순서대로 각 요소에 접근할 수 있습니다.

```javascript
let myMap = new Map();
```

| 함수                      | 정의                              |
| :------------------------ | :-------------------------------- |
| set(키, 값)               | 데이터(키-값) 저장                |
| get(특정 키)              | 특정 키에 해당하는 값 반환        |
| has(특정 키)              | 특정 키가 저장되어 있는지 확인    |
| delete(특정 키)           | 특정 키를 가지고 있는 데이터 삭제 |
| clear()                   | 모든 데이터 삭제                  |
| forEach(callback(데이터)) | 모든 데이터 읽기                  |

### 8. Math 객체

- Math 객체는 수학적인 상수와 내장 함수를 가진 객체로, 숫자 자료형만 지원합니다.

| 함수                       | 정의                                |
| :------------------------- | :---------------------------------- |
| Math.round(숫자)           | 숫자에 대한 반올림 처리             |
| Math.ceil(숫자)            | 숫자에 대한 올림 처리               |
| Math.floor(숫자)           | 숫자에 대한 내림 처리               |
| Math.trunc(숫자)           | 소수 부분은 버리고 정수 부분만 반환 |
| Math.pow(숫자, 제곱)       | 숫자^제곱 값 반환                   |
| Math.sqrt(숫자)            | 숫자의 루트값 반환                  |
| Math.abs(숫자)             | 숫자의 절대값 반환                  |
| Math.min(숫자1, 숫자2, ..) | 가장 작은 값 반환                   |
| Math.max(숫자1, 숫자2, ..) | 가장 큰 값 반환                     |
| Math.random()              | 0이상 1미만의 작은 숫자형 값 반환   |

### 9. JSON 객체

- JSON(JavaScript Object Notation)은 데이터를 저장하거나 전송할 때 많이 사용되는 DATA 교환 형식입니다.

| 함수                 | 정의                                                                           |
| :------------------- | :----------------------------------------------------------------------------- |
| JSON.stringify(객체) | 데이터를 서버로 전송하기 위해서는 데이터 형태를 문자열 형태로 변환해야 합니다. |
|                      | Object 데이터를 문자열로 변환해줍니다.                                         |
| JSON.parse(문자열)   | 서버로부터 응답받은 데이터는 문자열 형태입니다.                                |
|                      | 문자열을 Object 객체로 변환해줍니다.                                           |

```javascript
let data = {
  employees: [
    {
      firstName: "채은",
      lastName: "임",
    },
    {
      firstName: "지은",
      lastName: "이",
    },
    {
      firstName: "은우",
      lastName: "차",
    },
  ],
};
let stringValue = JSON.stringify(data);
```

```javascript
let text =
  '{"employees":[{"firstName":"채은","lastName":"임"},{"firstName":"지은","lastName":"이"},{"firstName":"은우","lastName":"차"}]}';
let obj = JSON.parse(text);
```

### 10. Window 객체

- window 객체는 전역 객체입니다.

| 함수                       | 정의                                               |

| setTimeout(함수, 시간)     | 지정한 시간 간격 이후에 정의한 함수 실행           |
| clearTimeout()             | setTimeout() 함수 실행 종료                        |
